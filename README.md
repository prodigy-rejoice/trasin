# Trasin

Translate documents across languages in seconds.

Trasin is an AI-powered document interpreter that allows users to upload PDFs or document images, translate them into their preferred language, and download the translated version while preserving structure and meaning.

## Features

* Upload PDF, JPG, and PNG documents
* Automatic language detection
* AI-powered translation with Gemini
* Download translated documents as PDF
* Clean and simple web experience

## Tech Stack

* Flutter Web
* Firebase
* Gemini (Firebase AI Logic)
* Firebase Hosting
